# EpiTrack

Many parents and caregivers struggle to properly track their children's epilepsy condition. Instead of caregivers relying on memory, EpiTrack provides a unified platform for tracking seizure events, medications, and adding notes and triggers to allow caregivers to work with healthcare providers to properly monitor seizures and accurately identify triggers to help subdue the impact of epilepsy on the child's life.

## 🧩 Problem

Parents and caregivers of children with epilepsy have difficulty keeping up with medication, tracking seizures, tracking environmental and physical triggers, and clearly communicating these activities to healthcare providers.

---

## 💡 Value

EpiTrack makes managing epilepsy and tracking seizures easier for caregivers by providing detailed documentation instead of relying on memory, helping identify triggers, and allowing healthcare providers to have a full and accurate picture to give the best care and treatment.

---

## 📋 Project Plan

My plan was to build a web application dashboard using HTML, CSS, and plain JavaScript, with no back end or database at this stage. The application gives caregivers the ability to create, read, update, and delete medications, medication dosages, seizure events, and triggers. I started by establishing the data structure and implementing the medication feature first, then built out seizure and trigger tracking, with all data stored locally in the browser on the client side.

---

## ✅ Features

### Complete
- Add, Edit, and Delete Medication
- Add, Edit, and Delete Medication Doses
- Add, Edit, and Delete Missed Medication
- Add, Edit, and Delete Seizure Events
- Add, Edit, and Delete Notes to Seizure Activity
- Add, Edit, and Delete Triggers
